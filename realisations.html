Mes réalisations professionnelles








Pendant mon BTS SIO, j'ai eu l'occasion de mener à bien des travaux pratiques, des travaux dirigés et des ateliers de professionnalisation. Voici quelques-unes de mes réalisations professionnelles.







Projet d’analyse de trafic




Site web en CSS/PHP permettant l'insertion et l'affichage de données extraites d'une base de données MySQL. Ces données sont issues d'un fichier de log répertoriant les connexions des utilisateurs aux différentes pages d'un site web.








Projet "Qui est-ce ?"




Site web en HTML/CSS/PHP simulant le jeu "Qui est-ce ?" et capable de deviner la personne choisie par le joueur, même si celui-ci ment à une seule question.








Projet CVVEN




Site web pour un village de vacances, développé en CSS/PHP avec le framework CodeIgniter. Ce site permet l'authentification des utilisateurs, ainsi que la création et la gestion des réservations, stockées dans une base de données MySQL.








Application CRUD




Application Java réalisée avec Swing, permettant d'effectuer les opérations CRUD (Création, Lecture, Mise à jour, Suppression) sur les données d'une table dans une base de données MySQL.








Installation d'un serveur GLPI




Installation, configuration et déploiement d'un serveur GLPI.








Reprise d'activité




Restauration d'une base de données GLPI suite à la perte du serveur à partir d'une sauvegarde.
